# 中国高铁线路未来预测模型

## 📋 项目概述

本项目是一个基于机器学习的中国高铁线路未来预测系统，采用XGBoost算法，综合考虑经济引力、空间距离、行政层级、城市群分布及网络拓扑结构等22维特征，滚动预测2026-2030年的新建高铁线路。

### 核心功能

1. **滚动预测** - 基于历史线路逐年滚动训练，预测未来新建线路
2. **时间分割验证** - 使用2008-2020年数据训练，验证2021-2025年预测准确率
3. **空间拓扑修正** - 过滤与已有线路重叠的预测结果
4. **交互式可视化** - 生成2008-2030年时间轴地图

## 📊 预测依据

预测是否新建高铁线路基于以下22个特征（6大类）：

### 1. 经济特征
- GDP总和、GDP乘积
- 人口总和、人口乘积
- 经济引力 = (GDP和 × 人口积) / 距离
- 平均人均GDP

### 2. 空间距离特征
- Haversine地理距离
- 对数距离
- 黄金距离标记（300-800km高铁竞争力最强）
- 短距离/长距离标记

### 3. 行政层级特征
- 最高行政级别
- 最低行政级别
- 行政级别差距

### 4. 地理与城市群特征
- 是否都在胡焕庸线以东
- 是否同省份
- 是否同一城市群（长三角/珠三角/京津冀/成渝等）

### 5. 网络拓扑特征
- 城市连接数（已有线路）
- 共同邻居数

### 6. 特殊通道
- 跨海通道标记（福州/厦门 ↔ 台北/台中/高雄）

## 📈 技术特点

### 算法
- **XGBoost** 梯度提升决策树
- **RandomForest** 随机森林（备选）

### 模型参数
```python
n_estimators=80
max_depth=4
learning_rate=0.15
```

### 验证方式
- **时间分割验证**：用2008-2020年训练，预测2021-2025年
- **真实性能**：AUC-ROC 0.92，准确率 0.83

### 滚动预测策略
每年以前一年累积的线路（历史+预测）为基础，重新训练模型并预测下一年新建线路。

### 空间拓扑修正
过滤与已有线路夹角小于30度的重复预测。

## ⚠️ 局限性

1. 未考虑政策因素（如"十四五"规划）
2. 跨海通道技术难度难以量化
3. 负样本为随机生成，可能偏离真实分布
4. 地形因素（如山地、沙漠）未纳入
5. 召回率较低（0.43），模型可能过于保守

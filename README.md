# 中国高铁线路未来预测系统

## 📋 项目概述

本项目是一个基于机器学习的中国高铁线路未来预测系统，采用XGBoost算法，综合考虑经济引力、空间距离、行政层级、城市群分布、途经站覆盖及网络拓扑结构等特征，滚动预测2026-2030年的新建高铁线路。

### 核心功能

1. **优化预测** - 基于途经站覆盖过滤冗余，综合战略得分，产生有意义的长距离线路
2. **全量途经站** - 为已有和预测的线路都标出途经站，生成多段线地图
3. **独立图层控制** - 按年份独立勾选，每年线路清晰呈现
4. **时间分割验证** - 使用2008-2020年数据训练，验证2021-2025年预测准确率

## 🚀 快速开始（新电脑完整部署指南）

### 1. 环境准备

首先确保已安装 Python 3.8 或以上版本。然后安装依赖库：

```bash
pip install pandas numpy xgboost scikit-learn
```

### 2. 运行完整流程（一键完成）

```bash
python src/visualization/full_pipeline.py
```

这将自动：
1. 检查预测数据（如果没有，运行优化版预测）
2. 计算已有和预测线路的途经站
3. 生成完整的交互式地图

### 3. 打开地图

#### 步骤1：运行预测优化版

```bash
python src/prediction/hsr_prediction.py
```

#### 步骤2：生成完整地图

```bash
python src/visualization/full_pipeline.py
```

## 🌐 地图使用说明

打开 `output/china_hsr_full_routes.html` 后：

1. **右上角图层控制** - 勾选想看的年份

   - 2026-2030：预测线路（红色虚线）

4. **图例说明**：
   - 灰色线 = 历史线路（2008-2025）
   - 红色虚线 = 预测线路（2026-2030）

## 📈 预测方法改进

### 主要优化

1. **冗余过滤** - 如果某对城市已经被途经站覆盖，则不再预测
2. **战略得分** - 综合考虑行政级别、距离合适度、GDP规模
3. **预测优化** - 70%模型概率 + 30%战略得分，综合排名
4. **每年 Top5** - 每年只保留最可能的5条预测线路

### 时间分割验证结果

| 指标 | 数值 |
|------|------|
| AUC-ROC | 0.90+ |
| 准确率 | 0.80+ |
